Database Request API

URL Request Type(s) Description
~/users/ Admin: GET Get all users
~/users/<user-id>/ Admin: GET Get a user by id
~/images/ Any: GET
Admin: POST		 Get all images or post a single image
~/images/<image-id>/ Any: GET
Admin: PUT, DELETE		 Get, update or delete an image by id
~/articles/ Any: GET
Admin: POST		 Get all articles or post a single article
~/articles/?title=&author=&body=&topic=&publication= Any: GET
Admin: POST		 Get articles by url parameter
~/articles/<article-id>/ Any: GET Get an article by id
~/articles/topic/<topic-id>/ Any: GET Get all articles connected to a topic
~/articles/publication/<publication-id>/ Any: GET Get all articles from a publication
~/articles/date/<year>/ Any: GET Get articles by year
~/articles/date/<year>/<month>/ Any: GET Get articles by month
~/articles/date/<year>/<month>/<day>/ Any: GET Get articles by day
~/articles/title/<query>/ Any: GET Search for articles by title grep
~/articles/body/<query>/ Any: GET Search for articles by body text grep
~/articles/author/<query>/ Any: GET Search for articles by author grep
~/topics/ Any: GET
Admin: POST		 Get all topics or post a single topic
~/topics/<topic-id>/ Any: GET
Admin: PUT, DELETE		 Get, update or delete a topic by id
~/publications/ Any: GET
Admin: POST		 Get all publications or post a single publication
~/publications/<publication-id>/ Any: GET
Admin: PUT, DELETE		 Get, update or delete a publication by id
